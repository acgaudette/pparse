```
Usage: pparse [option...]
  -i path       --input=path           file for reading
  -o path       --output=path          file for writing
  -n namespace  --namespace=namespace  namespace for output header
  -c container  --container=container  container for output class
  -L include    --include=include      using directive for output header
  -I name       --ignore=name          property name to ignore
  -C name       --cherry=name          container name to parse
```

## Example

```
pparse --include=Something --namespace=Message --ignore=is_the \
--cherry my_main_theme --cherry electric_age
```

### Input

```javascript
message = {
  'my main theme' : {
     'is the'         : true,
     'extension'      : [ 0.0, -0.5, 0.5],
     'of the'         : [ 0.5,    0,   1],
     'nervous system' : [ 0.0, -0.5, 0.5],
  },
  'electric age' : {
     'and thus'     : [ .09, .05, .4 ],
     'the complete' : [ .11, .02, .4 ],
  },
};
```
